PDFKit API: PDF 生成

使用 Node.js 的 PDFKit 库从文本内容生成 PDF 文档。

1. 从文本生成 PDF

端点: POST /generate-pdf

接收文本内容，生成 PDF 文件，并返回文件的访问路径。

请求体 (JSON):

JSON
{
  "text_content": "这是 PDF 文档的内容。\n可以包含多行文本。"
}

响应 (JSON):

JSON
{
  "message": "PDF generated successfully",
  "filePath": "/generated_pdfs/some_unique_filename.pdf" // 相对于 API 服务器的路径
}

生成的 PDF 文件可以通过拼接基础 URL 和 `filePath` 来访问，例如 `http://localhost:3009/generated_pdfs/some_unique_filename.pdf`。

如果发生错误，响应将包含错误信息：

JSON
{
  "error": "错误信息描述",
  "details": "可选的错误详情"
}
