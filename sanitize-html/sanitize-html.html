Sanitize HTML API: HTML 清理

使用 sanitize-html 库清理 HTML 输入，以防止 XSS 攻击。允许指定允许的标签和属性。

1. 清理 HTML 内容

端点: POST /sanitize-html

接收 HTML 内容和可选的清理配置，返回清理后的 HTML 字符串。

请求体 (JSON):

JSON
{
  "html_content": "<h1 style='color:blue;' onclick='alert(\"danger\")'>Test</h1><script>alert('evil')</script>",
  "options": { // 可选，参考 sanitize-html 文档获取所有选项
    "allowedTags": [ "h1", "p", "a", "strong" ],
    "allowedAttributes": {
      "h1": [ "style" ],
      "a": [ "href" ]
    },
    "allowedSchemes": [ "http", "https" ]
  }
}

如果未提供 options，则使用非常严格的默认设置（剥离所有标签和属性）。

响应 (JSON):

JSON
{
  "sanitized_html": "<h1 style=\"color:blue;\">Test</h1>"
}

如果发生错误，响应将包含错误信息：

JSON
{
  "error": "错误信息描述",
  "details": "可选的错误详情"
}
