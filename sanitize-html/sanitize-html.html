Sanitize HTML Content

Endpoint: /sanitize-html (POST)

Cleans HTML input to prevent XSS attacks. Allows specifying allowed tags and attributes.

Request Body:

{
    "html_content": "<h1>Hello <script>alert('XSS')</script>World</h1> <a href='javascript:alert(1)'>Click</a>",
    "options": { // Optional: See sanitize-html documentation for all options
        "allowedTags": [ "h1", "p", "a" ],
        "allowedAttributes": {
            "a": [ "href" ]
        }
    }
}

If no options are provided, a very restrictive default is used (strips all tags and attributes).

Example Usage (JavaScript):


fetch('http://localhost:3010/sanitize-html', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        html_content: "<h1 style='color:blue;' onclick='alert(\'danger\')'>Test</h1><script>alert('evil')</script>",
        options: {
            allowedTags: [ 'h1' ],
            allowedAttributes: { 'h1': [ 'style' ] }
        }
    }),
})
.then(response => response.json())
.then(data => console.log(data.sanitized_html))
.catch(error => console.error('Error:', error));
