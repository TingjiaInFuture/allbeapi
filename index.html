About AllBeAPI

A lightweight, universal SDK that integrates commonly used third-party libraries through a consistent API. This simplifies development by providing a single point of access for various functionalities, reducing the need to learn and manage multiple individual libraries and their dependencies.

Convert Markdown to HTML effortlessly using Marked.js integration.

Easy to Use SDKs

We provide SDKs in popular languages to get you started quickly.

JavaScript SDK Example

// Make sure to include allbeapi.js
// <script src="SDK/JavaScript/allbeapi.js"></script>

const apiClient = new AllBeApi();

// Example: Convert Markdown to HTML
apiClient.marked.render("# Hello AllBeAPI")
    .then(response => {
        console.log(response); // HTML output
        // Assuming the response is the direct HTML string or an object containing it
        // If it's an object like { html: "
Hello AllBeAPI
" }, access response.html
    })
    .catch(error => console.error("Error:", error));

// Example: Generate QR Code (returns a Blob)
apiClient.pythonQrcode.generateQrcode("https://res.allbeapi.top")
    .then(blob => {
        const imageUrl = URL.createObjectURL(blob);
        console.log("QR Code Image URL:", imageUrl);
        // You can then use this URL in an <img> tag
        // const imgElement = document.createElement('img');
        // imgElement.src = imageUrl;
        // document.body.appendChild(imgElement);
    })
    .catch(error => console.error("Error:", error));

Python SDK Example

# Make sure allbeapi.py is in your PYTHONPATH or same directory
# from allbeapi import AllBeApi # Assuming the file is named allbeapi.py

# For the provided structure:
# import sys
# sys.path.append('SDK/Python') # Adjust if running from a different root
# from allbeapi import AllBeApi


client = AllBeApi()

# Example: Convert Markdown to HTML
try:
    html_output = client.marked.render("# Hello AllBeAPI from Python")
    print(html_output) # This will likely be a JSON string or dict
except Exception as e:
    print(f"Error: {e}")

# Example: Generate QR Code (returns bytes)
try:
    qr_image_bytes = client.python_qrcode.generate_qrcode("https://res.allbeapi.top")
    with open("example_qrcode.png", "wb") as f:
        f.write(qr_image_bytes) # The response.content if using requests directly, or the direct bytes
    print("QR Code saved to example_qrcode.png")
except Exception as e:
    print(f"Error: {e}")
