Pygments API: 代码高亮

使用 Python 的 Pygments 库对代码进行语法高亮，并输出为 HTML。

1. 高亮代码

端点: POST /highlight

接收代码字符串和可选的语言名称，返回高亮后的 HTML 代码。

请求体 (JSON):

JSON
{
  "code": "def hello():\n  print(\"Hello, World!\")",
  "language": "python" // 可选，如果未提供，Pygments 会尝试自动检测语言
}

响应 (JSON):

JSON
{
  "highlighted_code": "<div class=\"highlight\"><pre><span>...</span>...</pre></div>\n",
  "detected_language": "python" // 如果语言是自动检测的，则包含此字段
}

如果发生错误，响应将包含错误信息：

JSON
{
  "error": "错误信息描述"
}
