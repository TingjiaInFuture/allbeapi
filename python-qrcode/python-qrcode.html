Python QR Code API: 二维码生成

使用 Python 的 qrcode 库生成二维码图像。

1. 生成二维码

端点: POST /generate-qrcode

接收生成二维码所需的数据和配置参数，返回二维码图像。

请求体 (JSON):

JSON
{ 
  "data": "https://www.example.com", // 必需，要编码到二维码中的数据
  "version": 1, // 可选，二维码版本 (1-40)，控制尺寸和数据容量
  "error_correction": "M", // 可选，纠错级别 ("L", "M", "Q", "H")
  "box_size": 10, // 可选，每个模块的像素数
  "border": 4, // 可选，边框宽度（模块数）
  "fill_color": "black", // 可选，填充颜色
  "back_color": "white", // 可选，背景颜色
  "format": "png" // 可选，输出图像格式 ("png", "jpeg")
}

响应 (Image):

成功时，直接返回生成的二维码图像文件 (例如 image/png 或 image/jpeg)。响应头中会包含 Content-Disposition 指示文件名，以及 Content-Type 表明图像类型。

如果发生错误，响应将是 JSON 格式：

JSON
{ 
  "error": "错误信息描述" 
}
