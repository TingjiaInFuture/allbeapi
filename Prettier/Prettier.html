Prettier API: 代码格式化

使用 Prettier 自动格式化您的 JavaScript, HTML, CSS, Markdown 等多种语言的代码，保持一致的代码风格。

1. 格式化代码

端点: POST /prettier/format

接收代码字符串、解析器名称以及可选的 Prettier 配置，返回格式化后的代码。

请求体 (JSON):

JSON
{
  "code": "const foo = {bar:'baz', arr: [1,2,3]}; function hello(){console.log('world')}",
  "parser": "babel", // 必需，例如 'babel', 'typescript', 'html', 'css', 'markdown', 'json' 等
  "options": { // 可选，Prettier 配置选项
    "singleQuote": true,
    "semi": false,
    "printWidth": 100
  }
}

响应 (JSON):

JSON
{
  "formattedCode": "const foo = { bar: 'baz', arr: [1, 2, 3] }\nfunction hello() {\n  console.log('world')\n}
",
  "success": true
}

如果格式化失败，success 将为 false 并包含错误信息。

2. 检查代码格式

端点: POST /prettier/check

检查提供的代码是否已经符合 Prettier 的格式化标准。

请求体 (JSON):

JSON
{
  "code": "const foo = { bar: 'baz' };",
  "parser": "babel",
  "options": { // 可选
    "singleQuote": true
  }
}

响应 (JSON):

JSON
{
  "isFormatted": true, // 如果代码已格式化则为 true，否则为 false
  "success": true
}

3. 获取支持的解析器列表

端点: GET /prettier/parsers

返回 API 支持的所有解析器及其对应的文件扩展名。

响应 (JSON):

JSON
{
  "success": true,
  "parsers": {
    "javascript": "babel",
    "js": "babel",
    "jsx": "babel",
    "typescript": "typescript",
    "ts": "typescript",
    // ...更多解析器
  },
  "supportedExtensions": ["javascript", "js", "jsx", "typescript", "ts", /* ... */]
}

4. 获取默认/可用配置选项

端点: GET /prettier/options

返回 Prettier 的默认配置选项以及可用的配置项及其说明。

响应 (JSON):

JSON
{
  "success": true,
  "defaultOptions": {
    "printWidth": 80,
    "tabWidth": 2,
    // ...更多默认选项
  },
  "availableOptions": {
    "printWidth": "number - 每行最大字符数",
    "tabWidth": "number - 制表符宽度",
    // ...更多可用选项及其说明
  }
}

5. API 信息

端点: GET /prettier/info (或 GET /)

提供关于 Prettier API 的详细信息，包括服务描述、版本、所有端点及其功能、支持的语言和使用示例。

响应 (JSON):

JSON
{
  "success": true,
  "service": {
    "name": "Prettier API",
    "description": "Code formatting service using Prettier",
    "version": "x.y.z" // Prettier API 版本
  },
  "endpoints": {
    "POST /prettier/format": "Format code using Prettier",
    // ...其他端点信息
  },
  "supportedLanguages": ["javascript", "typescript", /* ... */],
  "examples": {
    "format": { /* ... */ }
  }
}
